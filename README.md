# Expense Tracker Web App
A clean, interactive expense tracking application built with **Streamlit**, designed to help users record, analyze, and visualize their spending habits.

# Features

*  Add and manage daily expenses
*  Automatic calculation of total spending
*  Categorize expenses (Food, Transport, School, etc.)
*  Visual insights with charts:

* Category-wise spending
* Monthly spending trends
*  Persistent storage using CSV
*  Web-based interface (accessible from any device)


# Tech Stack

* **Python**
* **Streamlit** – UI & interactivity
* **Pandas** – Data handling
* **Matplotlib** – Data visualization

# Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/GbosiSophia/expense-tracker.git
cd expense-tracker
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app locally:

```bash
streamlit run expense_tracker_web.py
```

## 🌍 Deployment (Render)

1. Push your project to GitHub
2. Create a **Web Service** on Render
3. Use the following start command:
